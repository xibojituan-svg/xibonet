# 学员订单信息查询

# 一、查询某一时间段内下单的学员列表

请求url：https://ops.xibojiaoyu.com/xmkp-backend-middle/ops/diverseInfo/orders/queryByTimeRange

请求形式：Post

入参定义

| 入参定义 | 类型 | 是否必填 | 字段含义 | 备注 |
| --- | --- | --- | --- | --- |
| startTime | Long | *   [x] | 下单查询开始时间 | 时间戳（毫秒）<br>开始、结束时间间隔不得超过7天 |
| endTime | Long | *   [x] | 下单查询结束时间 |
| orderType | Integer | *   [ ] | 订单类型 | \-1：查询全部订单（包含体验营、正价营）<br>1：查询体验营订单<br>2：查询正价营订单 |
| state | Integer | *   [ ] | 订单状态 | \-1：查询全部状态订单（包含已支付、已退款）<br>0：查询已支付订单<br>1：查询已退款订单 |

入参示例

```plaintext
{
    "startTime": 1764596818000,
    "endTime": 1764683218000,
    "orderType": -1,
    "state": -1
}
```

| 返回值定义 |  | 类型 | 字段含义 | 备注 |
| --- | --- | --- | --- | --- |
| code |  | int | 状态码 | 0：成功， 非0失败 |
| data |  | List<Long> | 学员id集合 |  |
| message |  | String |  | 非0时输出失败信息 |
| success |  | int |  | 跟code类似，使用code判断即可 |

返回值成功示例

```plaintext
{
    "message": "正常",
    "code": 0,
    "data": [
        676556437,
        232386878,
        442948967
    ]
}
```

返回值失败示例

```plaintext
{
    "message": "查询时间范围不能超过7天",
    "code": 201,
    "ret": -1,
    "data": null,
    "timestamp": 1775127827611
}
```

# 二、查询学员订单基础信息

请求url：https://ops.xibojiaoyu.com/xmkp-backend-middle/ops/diverseInfo/studentOrders/query

请求形式：Post

入参定义

| 入参定义 | 类型 | 是否必填 | 字段含义 | 备注 |
| --- | --- | --- | --- | --- |
| studentUids | List<Long> | *   [x] | 学员uid集合 | 一次传入数量不可超过100个 |
| orderStartTime | Long | *   [x] | 下单查询开始时间 | 时间戳（毫秒）<br>开始、结束时间间隔不得超过7天 |
| orderEndTime | Long | *   [x] | 下单查询结束时间 |
| orderType | Integer | *   [ ] | 订单类型 | \-1：查询全部订单（包含体验营、正价营）<br>1：查询体验营订单<br>2：查询正价营订单 |
| state | Integer | *   [ ] | 订单状态 | \-1：查询全部状态订单（包含已支付、已退款）<br>0：查询已支付订单<br>1：查询已退款订单 |

入参示例

```plaintext
{
    "studentUids": [676556437,232386878],
    "orderStartTime": 1764596818000,
    "orderEndTime": 1764683218000,
    "orderType": 1,
    "state":-1
}
```

| 返回值定义 |  | 类型 | 字段含义 | 备注 |
| --- | --- | --- | --- | --- |
| code |  | int | 状态码 | 0：成功， 非0失败 |
| data |  |  |  |  |
|  | orderNo | String | 订单号 |  |
|  | spuFulfillOrderNo | String | 订单履约号 |  |
|  | studentUid | Long | 学员UID |  |
|  | orderType | Integer | 订单类型：1-体验营，2-正价营 |  |
|  | semesterId | Long | 学期ID |  |
|  | courseId | Long | 训练营营期ID |  |
|  | amount | String | 订单金额 |  |
|  | orderState | String | 订单状态：0-正常，1-已退款 |  |
|  | createTime | Long | 下单时间 |  |
| message |  | String |  | 非0时输出失败信息 |
| success |  | int |  | 跟code类似，使用code判断即可 |

返回值成功示例

```plaintext
{
    "message": "正常",
    "code": 0,
    "ret": 0,
    "data": [
        {
            "orderNo": "202512014001437007939119355",
            "spuFulfillOrderNo": "2025120130004370043505743730",
            "studentUid": 676556437,
            "orderType": 1,
            "semesterId": 24068,
            "courseId": 1113,
            "amount": "9.9",
            "orderState": "0",
            "createTime": 1764596835000
        }
    ],
    "timestamp": 1775198940628
}
```

返回值失败示例

```plaintext
{
    "message": "查询时间范围不能超过7天",
    "code": 201,
    "ret": -1,
    "data": null,
    "timestamp": 1775128420908
}
```
