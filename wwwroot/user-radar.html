📐 评分维度说明

⚡ 执行力指数
作业完成率 + 打卡频率 + 主动行为 + 凌晨录音 + 帮助他人意愿
💰 变现紧迫度
经济压力 + 负债情况 + 开源迫切程度 + 期望收入与现状落差
🌐 社群影响力
群内发言量 + 帮助他人 + 被@频率 + KOL特质 + 转介绍潜力
📈 LTV潜力值
续费概率 + 客单价承受力 + 转介绍潜力 + 内容共创价值综合评分
🛡️ 心理脆弱度
自我怀疑信号 + 沉没成本焦虑 + 退费暗示 + 外部施压（家庭反对）
💎 经济投入度
总付费金额 ÷ 估算月收入 = 投入比（越高说明沉没成本越重）
